# Customer Shopping Behavior Analysis

## Overview

This project focuses on analyzing customer shopping behavior using Python, SQL, and Power BI. The objective is to uncover purchasing patterns, customer preferences, and key business insights through data analysis and visualization.

The project covers the complete analytics workflow, including data cleaning, exploratory data analysis (EDA), SQL-based querying, dashboard development, and business reporting.

---

## Dataset

The dataset contains customer shopping information, including:

* Customer demographics
* Product categories
* Purchase amounts
* Payment methods
* Shopping frequency
* Discounts and promotions
* Customer reviews and ratings

The data was analyzed to identify trends, customer behavior patterns, and business opportunities.

## Project Workflow

### 1. Data Loading

## How to Run

### Python Analysis

1. Clone the repository

```bash
git clone <repository-link>
```

2. Install required libraries

```bash
pip install pandas
```

3. Open Jupyter Notebook

```bash
jupyter notebook
```

4. Run the notebook

```bash
customer_shopping_behavior-checkpoint.ipynb
```

### SQL Analysis

1. Import the dataset into PostgreSQL.
2. Execute the SQL queries provided in:

```text
Customer Behavior Analysis SQL Queries.sql
```

### Power BI Dashboard

